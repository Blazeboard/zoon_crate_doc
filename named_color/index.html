Module zoon::named_color

source · [−]
Expand description

Set of colors available for use in Zoon. They can be used in every methods accepting HSLuv colors. The palette is based on https://tailwindcss.com/docs/customizing-colors

Example

use zoon::{named_color::*, *};

let (hovered, hovered_signal) = Mutable::new_and_signal(false);

let button = Button::new()
    .s(Background::new().color_signal(hovered_signal.map_bool(|| GREEN_7, || GREEN_8)))
    .on_hovered_change(move |is_hovered| hovered.set_neq(is_hovered));
